오사카 우메다 쇼핑몰 쿠폰

오사카역 우메다 지역 6곳의 쇼핑센터에서 쇼핑이나 식사에 사용할 수 있는 1000엔 쿠폰(500엔X2장)입니다.

쿠폰 사용 안내

사용방법
  • ✔ 쿠폰 파일, 또는 이미지를 저장해주세요!
  • ✔ 쿠폰과 여권을 안내데스크에서 제시해 주세요.
유의사항
  • ✔ 1인당 1회 사용 가능, 방일 외국인 대상.
  • ✔ 예고 없이 내용이 변경, 또는 종료될 수 있음
  • ✔ 쿠폰을 나눠드리는 곳은 4곳이지만, 6개 시설에서 모두 사용하실 수 있습니다. (쿠폰 교환 가능처 : GRAND FRONT OSAKA 북관 1층, 한큐 삼번가 남관 지하 1층, HEP FIVE 1층,HERBIS PLAZA ENT 지하 2층)
  • ✔ 3000엔 이상 구매하시는 경우에 한해, 500엔 혜택을 받을 수 있음.
이용가능 시설
  • ✔ GRAND FRONT OSAKA
  • ✔ 한큐 삼번가
  • ✔ NU chayamachi/NU chayamachi PLUS
  • ✔ HERBIS PLAZA/HERBIS PLAZA ENT
  • ✔ 디아모르 오사카
